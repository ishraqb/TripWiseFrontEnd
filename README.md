# TripWise

A simple web version of my TripWise trip planner. Enter your trip details and it
shows three plans: Cheapest, Balanced, and Experience-Focused.

Built with plain HTML, CSS, and JavaScript.

## How to run

Open `index.html` in a web browser.

## Files

- `index.html` - the page and the trip form
- `style.css` - the styling
- `app.js` - reads the form and builds the three plans
